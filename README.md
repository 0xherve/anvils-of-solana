# ANVL SPL-Token

This project is for creating and minting a token called `ANVL` on the Solana devnet cluster using the `gill` solana client library. Afterwhich it mints `1000 ANVL` tokens to `EmiNAMemxkFNcq252pccGk37QytVNXNRsV48KcRFPYs5` on Solana Devnet. 


## Scripts

There are two scripts in this project:
 -`ace_token_mint.ts`, which creates and mints a new token on the Solana devnet.
 -`mint_tokens.ts`, which creates an Associated Token Address, and mints 1000 ANVL tokens to it
 
To run the script, execute the following command:

```bash
node ace_token_mint.ts
node mint_tokens.ts
```
